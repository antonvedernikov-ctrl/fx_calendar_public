fx_calendar_public

Public mirror of EA-consumable calendar artifacts. Updated daily at ~23:15 UTC.

Artifacts

FileDescriptionFormat
calendar/events.parquet Merged economic calendar (FF primary + MQL5 secondary), historic + 60-day forward Apache Parquet
calendar/holidays.parquet Consolidated Tier 1 + Tier 2 bank holidays Apache Parquet
calendar/metadata.json Pipeline metadata: row counts, coverage range, generated_at_utc JSON
features/seasonality_daily.parquet Deterministic daily seasonality features Apache Parquet

Schema

See schema.json for field-level schema of each artifact.

Consumption examples

curl (no auth required)

curl -L -o events.parquet \
  https://antonvedernikov-ctrl.github.io/fx_calendar_public/calendar/events.parquet

Python

import io, requests, pandas as pd

url = "https://antonvedernikov-ctrl.github.io/fx_calendar_public/calendar/events.parquet"
r = requests.get(url, timeout=30)
r.raise_for_status()
df = pd.read_parquet(io.BytesIO(r.content))
print(df[df.is_future].head())

MQL5 WebRequest

// Tools → Options → Expert Advisors → Allow WebRequest for:
//   https://antonvedernikov-ctrl.github.io
string url = "https://antonvedernikov-ctrl.github.io/fx_calendar_public/calendar/metadata.json";
char   post[], result[];
string headers;
int    timeout = 5000;
int    code = WebRequest("GET", url, "", timeout, post, result, headers);
if (code == 200) {
   string body = CharArrayToString(result, 0, WHOLE_ARRAY, CP_UTF8);
   Print("metadata: ", body);
}

Staleness check

curl -sf https://antonvedernikov-ctrl.github.io/fx_calendar_public/calendar/metadata.json \
  | jq '.generated_at_utc'

If generated_at_utc is older than 36 hours, the data may be stale.
